# KOVA Finance | Collective Wealth. Redefined.

KOVA is a premium, secure, and transparent co-operative financial management platform. Built for modern societies, KOVA streamlines savings, loan lifecycles, and logistics management with a tier-1 banking aesthetic.

## 🚀 Key Features

- **Geometric Design System**: A premium "Midnight Navy and Growth Emerald" design language.
- **Role-Based Access**: Specialized portals for Members, Secretaries, and Administrators.
- **Loan Lifecycle**: Automated applications, interest calculations, and repayment tracking.
- **Targeted Savings**: Flexible goal-based savings for members.
- **Logistics & Foodstuffs**: Integrated management for society inventory and deductions.
- **Audit & Security**: Full audit trails and secure payment verification workflows.

## 🛠️ Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org) (App Router, Turbopack)
- **Database**: PostgreSQL via [Supabase](https://supabase.com)
- **ORM**: [Prisma](https://prisma.io)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com)
- **Email**: [Resend](https://resend.com)

## 🚦 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Environment Setup**:
   Create a `.env` file based on `.env.example` (if available) with your Supabase and Prisma credentials.

3. **Database Migration**:
   ```bash
   npx prisma db push
   ```

4. **Run Development Server**:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to view the portal.
